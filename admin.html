🗄 Setting Up Google Sheets as Admin Dashboard

For production use, all 50+ learner results are stored in Google Sheets. Follow these steps:

  1. Create a new Google Sheet named CogniAssess Results
  2. Open Extensions → Apps Script
  3. Paste the code from apps-script-code.gs into the editor
  4. Click Deploy → New Deployment → Web App
  5. Set Execute as: Me and Who has access: Anyone
  6. Copy the Web App URL and paste it into storage.js → WEB_APP_URL
  7. In Google Sheets, use Filter Views to filter by Batch, Score, or Level
  8. Use Freeze Row 1 so headers stay visible while scrolling
Concurrency Note: Google Apps Script handles ~30–50 concurrent submissions reliably using LockService. For 100+ simultaneous users, migrate to Firebase Firestore or Supabase for better throughput.
